# Sayan Mondal — Portfolio

Personal portfolio and research archive for Sayan Mondal. The site is plain HTML, CSS, and JavaScript with no build dependencies.

## Local preview

Open `index.html` directly, or serve the repository with any static web server.

## Deployment

`.github/workflows/pages.yml` deploys the repository to GitHub Pages on every push to `master`. In the repository settings, set **Pages → Build and deployment → Source** to **GitHub Actions**.
